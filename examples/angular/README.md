# Angular Example

Using jsPDF in Angular is really simple with only two steps

### 1.  Install jsPDF

`npm i jspdf`

### 2.  Import jsPDF in the component file

```ts
import { jsPDF } from "jspdf"
```
And now you can use it within that component

<br>

## To run this example

### 1. Install all the dependencies
`npm i`

### 2. Run 
`npm start`
